# Development

## Getting Started

Clone the repository:

```bash
$ git clone https://github.com/JrGoodle/clowder.git
$ cd clowder
```

## Building

Install and uninstall scripts:

```bash
$ ./install.sh
$ ./uninstall.sh
```

## Testing

Run unit tests.

```bash
$ ./scripts/unittests.sh 
```

## Updating version

```bash
$ ./update_version.sh $OLD_VERSION $NEW_VERSION
```
